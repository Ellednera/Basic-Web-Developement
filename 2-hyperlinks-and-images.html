Hyperlinks and Images

Hyperlinks

Hyperlinks, also known as links, are used to link/bring the users on a web page to another web page. By default, hyperlinked texts are blue in colour before it is cliked. Once clicked, it will turn to purple colour. The clicked links will remain purple until you clear your search history in the web broswer. The hyperlinked text will turn red if the hyperlinked text is active. Below is an example of a hyperlink.

Go to Google.

Attributes for Hyperlink Tag

In the hyperlink tag, there are 2 attributes that are:
1. href="value"
2. target="value"

How to Use the Hyperlink Tag

This is how a hyperlink which brings the user to http://www.google.com is written.

<p>
Go to <a href="http://www.google.com" target="_blank">Google </a>
</p>

1. <a></a> is the hyperlink tag.
2. The href attribute tells the web browser where to go after the hyperlinked text is clicked.
3. The target attribute tells the web browser how and where to display the web page which is linked to.

NOTES
1. The value for the href attribute is either a file destination or a website. For websites, the value must have "http://" or else the web browser will think that it is a file destination and searches for the file in your directory. To link to another web page in a file in the same directory, declare the filename TOGETHER WITH ITS EXTENSION. To link to a web page inside of a different directory, declare the name of the directory and put a forward slash(/) until you reach the file you want to link to.

2. The easiest value to use in the target attribute is _blank, this will display the linked web page in a new tab. If you want the linked web page to display in the original web page, there is no need to use the target attribute.


Images

In a web page, you can also insert images by using the image tag, <img>. An image tag is an empty element and doesn't need to be closed.

When inserting an image, you might want to resize or declare the size of your image. This can be done by using the width and height attributes. The values of these two attributes are numbers, e.g. height="300". The web broswer will reconise the number as the number of pixels.

If you would like to changes the size of an image but maintain its shape, this is when Mathematics comes in. You will need to calculate the size of the image using ratio. Below is the formula: 

Let original width=xo
Let original height=yo
Let new width=xn
Let new height=yn

xo/yo = xn/yn

If your final answer is a double(number with decimals) like 367.39, just round it off to the nearest integer. One pixel is no big deal as a pixel is very small in size(smaller than 1mm).
