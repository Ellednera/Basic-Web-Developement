Introduction

HTML is the short form for Hyper Text Markup Language. HTML is not a programming language, it is only a markup language which tell the web browser how to present your data. HTML is the fundamental language for building/creating web pages. The current version of HTML is version 5(HTML5) which we are using.

Website/Web page was created by Sir Tim Bernes-Lee(Sir Timothy John Bernes-Lee). Before web pages exists, scientists needed to log in to different computers to retrieve certain data becuase most of the data was stored in different computers. This is why Sir Tim Bernes-Lee created web pages. With a web page, all the data can be stored in one place and can be retrieved easily through the Internet.

All languages for building web pages, for example HTML and CSS are controlled by the World Wide Web Consortium (W3C) in order to provide standard universal languages.

Markup languages consist of YAML, XML, HTML, CSS and XHTML while programming languages used for building web pages include Javascript(not to be confused with Java), PHP, SQL, JQuery, Perl, Ruby and SASS.

Before we start with HTML, there are some important things to take note of.

  • You will need a better text editor like Notepad++. However, the built-in text editor in Windows operating systems can still be used. The advantages of using Notepad++ is that it has syntax-highlighting features to ease your job.
  • Whenever you save a new file, a pop-up box will appear asking for the destination of the file you want to save. This box is called a dialoge box.
  • There are 2 extensions for HTML files, .htm(for very old Windows systems which only supports 3 characters for extensions) and .html

Let's Begin

Every HTML file starts with a declaration of the document type at the very first line. The declaration in a HTML file tells the web broswer what version of HTML we are using. In HTML version 5, the declaration is very simple, just type in <!DOCTYPE html> and the browser will know that we are using HTML5.

After declaring the document type here comes the html tag,<html></html>. All the contents of your web page goes in here.

Inside the html tag, there are two important elements(tags), the head tag,<head></head> and the body tag, <body></body>.

The head tag functions as the brain for your web page while the body tag comsists of all the data you want to print on the screen.

For beginners, you will only use the title tag,<title></title> inside of the head tag. The title tag must go inside of the head tag. The title tag represents the title for your HTML file. It is also used in displaying the title of a website in search engines.

In HTML files, anything(except in the paragraph tag) with the word "title" will display a small box with the same text when you point your pointer to the text. Any text within the title tag will display the text at the tab of a page. Now, move your pointer to the tab and you will see how the title tag works.

Let's Print Some Text!

To print text, use the paragraph tag, <p></p>.

To make headings, use the heading tag, <h1></h1>.
In HTML, the size of the heading is limited, which is from h1 to h6 only. h1 prints a heading with the biggest text while h6 prints the smallest text for a heading. Below is a example for the paragraph tag and the heading tag.
