Lists

Lists are used to list out ceratin items. In HTML, there are 3 types of lists, ordered list, unordered list and description list. Below are examples of how these 3 lists are used.

An ordered list uses the tag <ol></ol>.
An unordered list uses the tag <ul></ul>
an description list uses the tag <dl></dl>

To list out items in an ordered and unordered lists, use the list item tag,<li></li>.

To list out items in a description list, use this tag,<dt></dt> to define the terms(heading). To define the terms, type <dd></dd> after each term.

Below is some examples of how these lists are written and used.

1. Ordered List


Code:


<ol>
<li>Strawberry</li>
<li>Raspberry</li>
<li>Craneberry</li>
<li>Blueberry</li>
<li>Mulberry</li>
</ol>

The above code will display the following:

  1. Strawberry
  2. Raspberry
  3. Craneberry
  4. Blueberry
  5. Mulberry

2. Unordered List


Code:


<ul>
<li>Strawberry</li>
<li>Raspberry</li>
<li>Craneberry</li>
<li>Blueberry</li>
<li>Mulberry</li>
</ul>

The above code will display the following:

  • Strawberry
  • Raspberry
  • Craneberry
  • Blueberry
  • Mulberry

*By default, the shape of the bullets is a circle. Trust me, the bullets are circle in shape. Just zoom in and you shall see.

3. Description List


Code:


<dl>

<dt>Water</dt>
<dd>
Consists of 2 hydrogen atoms and 1 oxygen atom. Together they form a covalent compound of H<sub>2</sub>O.
</dd>

<dt>"The Art of War" by Tsun Wu</dt>
<dd>
Teaches one how to battle against one's enemies during wars in the oldern days. Nowadays, it is used as a bussiness strategy.
</dd>

<dt>Hot Dog</dt>
<dd>
A type of food which a sausage is sandwiched by a bun, usually has tomato sause and some mustard as the topping.
</dd>

</dl>

The above code will display the following:

Water
Consists of 2 hydrogen atoms and 1 oxygen atom. Together they form a covalent compound of H2O.
"The Art of War" by Tsun Wu
Teaches one how to battle against one's enemies during wars in the oldern days. Nowadays, it is used as a bussiness strategy.
Hot Dog
A type of food which a sausage is sandwiched by a bun, usually has tomato sause and some mustard as the topping.
