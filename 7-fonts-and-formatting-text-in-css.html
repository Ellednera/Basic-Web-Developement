Fonts

Fonts are used to tell the web browser what type of text to print. In web developement, fonts are divided into two groups, that are generic family and font family. The generic family is like the "gene" of a font while the font family contains the exact name of certain fonts, for example Times New Roman, MV Boli and any other fonts that can be found in Microsoft Words or in any other programmes similar to it.

Usually, we define the font of certain text with the font family followed by the generic family just in case the web browser does not support the font family. This is because not all the computers have the same font names. The web browser compares the font family with the fonts available on the clients' computer. When the font family is not available, the web browser will use the next font or the generic family to format the text.

There are four generic families:

  1. Serif(e.g. Times New Roman)
  2. Sans-serif(e.g. Arial)
  3. Monospace(e.g. Lucida Console)
  4. Cursive(e.g. MV Boli)
