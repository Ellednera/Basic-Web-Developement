CSS

CSS stands for Cascade/Cascading Style Sheets. The current version is version 3, and it can be written as CSS3.

What are the Uses of CSS?

CSS is used to decorate your HTML pages. It is like putting a layer of flesh, skin, clothes as well as spraying some perfume on a skeleton of a human body.

With CSS, you can do more amazing things that are not supported in HTML, for example making a hyperlink change its colour when your pointer is pointing at it, highlight text when the pointer points at it, changing the colour of some words in a sentence, so on and so forth.

How CSS Scripts Look

Where Should CSS Scripts Be Put?

There are 3 ways to use CSS in your HTML pages:

1. Inline: Using the style attribute, style="value"

2. Internal: Using the style tag, <style></style>, inside of the head, <head></head>, tag

3. External: Using a link tag, <link rel="value" href="destination">, to link CSS scripts from a different place

The Structure of a CSS Syntax

A CSS syntax consists of a SELECTOR and a DECLARATION.

Here is an example of a CSS syntax(command):


p {color: black;}

1. p is a selector. A selector defines which HTML tag it is formatting/pointing to.

2. color: black; is a declaration. A declaraiton defines how the text in the <p> tag should be formatted.

3. A declaration consists of a PROPERTY and a VALUE. In the example above, color is a property while black is the value for the color property.

NOTES
1. The property and the value of the property is separated by a colon(:)
2. Every declaration ends with a semicolon(;)
3. A pair of curly braces/brackets, {} is used after the selector and all the declarations are written in the pair of curly brackets.
4. Each property in CSS has its own values.

Now, let's rest for a while.

CSS is like a high-level programming language and all syntaxes/commands must end with a semicolon or else the command won't work.
