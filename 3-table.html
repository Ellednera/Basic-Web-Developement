In HTML, a table consists of several different elements(tags):
1. table <table> </table>
2. table heading <th> </th>
3. table row <tr> </tr>
4. table data <td> </td>

For beginners, you will learn only the "border" attribute. This attribute supports only numbers ranging from 0 to 10.
The border attribute is written as: border="value"

In some broswers, the default(original) value the border attribute is 0. This means that there is no lines to surround the table.
When the value for the border attribute is 0, this is what will happen:
